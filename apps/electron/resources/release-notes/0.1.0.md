# Mortise 0.1.0

The first release in Mortise's independent version line.

## Improvements

- **Independent product identity** - Application, package, protocol, configuration, and distribution identifiers now belong to Mortise and no longer track the former upstream release line.
- **Data sources start disabled** - New and unset installations keep the optional data-source UI and session tools off until explicitly enabled in Settings.
- **Workspace activity at a glance** - Workspace rows in the sidebar show a running indicator whenever one of their visible sessions is processing, even when another workspace is selected.

## Bug Fixes

- **Stop now truly interrupts a response** - Mortise waits for Pi to acknowledge cancellation, suppresses late output from the stopped turn, and no longer copies submitted or queued prompts back into the composer.
